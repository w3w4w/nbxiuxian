参数说明

- hpbuff: 提升气血百分比
- mpbuff: 提升真元百分比
- atkbuff: 提升攻击力百分比
- ratebuff: 提升修炼速度百分比
- crit_buff: 提升会心率百分比
- def_buff: 提升减伤率百分比
- dan_exp: 提升炼丹经验
- dan_buff: 提升炼丹产出丹药数量
- reap_buff: 提升药材收取数量
- exp_buff: 突破失败时提供经验保护百分比
- critatk: 提升会心伤害百分比
- two_buff: 增加双修次数
- number: 提升突破概率百分比
- clo_exp: 提升闭关经验百分比
- clo_rs: 提升闭关生命回复百分比
- random_buff: 战斗时随机获得一个战斗属性
- ew: 使用特定物品时伤害增加百分比
- desc: 功法的描述信息
- rank: 功法的品质等级
- level: 功法的等级要求

特殊技能参数说明

- skill_type: 技能类型（1-攻击，2-持续伤害，3-增强自身，4-封印，5-随机伤害，6-叠加伤害，7-变化神通）
- atkvalue: 攻击倍数
- turncost: 释放技能后的回合休息
- rate: 释放概率百分比
- bufftype: 增强自身的类型（1-攻击力，2-减伤率）
- buffvalue: 增强自身的倍数或百分比
- success: 封印技能的命中成功率百分比
- hpcost: 消耗当前血量的百分比
- mpcost: 消耗真元的百分比
- stone: 提升boss战灵石获取百分比
- integral: 提升boss战积分获取
- jin: 禁止对手吸取
- drop: 提升boss掉落率
- fan: 使对手发出的debuff失效
- break: 穿甲百分比
- exp: 增加战斗获得的修为
- ew_name: 使用特定物品时的名称

身法和瞳术参数说明

- buff_type: 类型（1-闪避率，2-命中率）
- buff: 提升闪避率或命中率的百分比
- buff2: 第二个百分比值（用于范围）

防具参数说明

- def_buff: 提升减伤率百分比
- atk_buff: 提升攻击力百分比
- crit_buff: 提升会心率百分比

法器参数说明

- atk_buff: 提升攻击力百分比
- crit_buff: 提升会心率百分比
- critatk: 提升会心伤害百分比
- def_buff: 提升减伤率百分比
- mp_buff: 降低真元消耗百分比
- zw: 装备专属武器时提升伤害
- name: 法器的名称
- level: 法器的品阶
